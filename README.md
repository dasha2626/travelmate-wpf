# TravelMate

TravelMate to aplikacja desktopowa wspomagająca planowanie, organizację oraz dokumentowanie podróży.

## Funkcjonalności

- 🗺️ Interaktywna mapa (GMap.NET) – wizualizacja odwiedzonych i planowanych miejsc
- ✈️ Wyszukiwarka lotów – przeglądanie połączeń lotniczych z rankingiem i trasami na mapie
- 🏨 Zarządzanie noclegami i dojazdami – rejestrowanie szczegółów podróży
- 💰 Kontrola wydatków – analiza kosztów z przeliczaniem walut (Frankfurter API)
- 📅 Kalendarz podróży – przegląd wyjazdów według dat
- 📊 Statystyki – łączne kilometry, liczba krajów, miast, lotów, średnia ocena
- 📄 Eksport do PDF – raporty z podsumowaniem podróży
- 🤖 Rekomendacje AI (Gemini 2.5 Flash) – propozycje kierunków na podstawie preferencji, budżetu i dat
- 📍 Punkty POI (OpenTripMap API) – wyszukiwanie atrakcji turystycznych w wybranym mieście
- 🛫 Lotniska europejskie (AirportsAPI) – wizualizacja lotnisk na mapie
- 🌍 Podświetlanie odwiedzonych krajów na mapie

## Technologie

- C# / WPF / .NET Framework 4.7.2
- GMap.NET
- SQLite / Entity Framework 6
- Gemini API, OpenTripMap API, Frankfurter API, AirportsAPI, Nominatim

## Instalacja i uruchomienie
1. Rozpakuj folder Release ZIP
2. Uruchom plik .exe znajdujący się w rozpakowanym folderze


# TravelMate EN

TravelMate is a desktop application for planning, organizing, and documenting trips.

## Features

- 🗺️ Interactive map (GMap.NET) – visualization of visited and planned destinations
- ✈️ Flight search – browsing flight connections with ranking and routes on the map
- 🏨 Accommodation & transport management – recording travel details
- 💰 Expense tracking – cost analysis with currency conversion (Frankfurter API)
- 📅 Travel calendar – browsing trips by date
- 📊 Statistics – total kilometers, number of countries, cities, flights, average rating
- 📄 PDF export – reports with trip summaries
- 🤖 AI recommendations (Gemini 2.5 Flash) – destination suggestions based on preferences, budget and dates
- 📍 POI search (OpenTripMap API) – finding tourist attractions in a selected city
- 🛫 European airports (AirportsAPI) – airport visualization on the map
- 🌍 Visited countries highlight on the map

## Technologies

- C# / WPF / .NET Framework 4.7.2
- GMap.NET
- SQLite / Entity Framework 6
- Gemini API, OpenTripMap API, Frankfurter API, AirportsAPI, Nominatim

## Installation & Setup

1. Extract the Release ZIP folder
2. Run the `.exe` file located in the extracted folder
