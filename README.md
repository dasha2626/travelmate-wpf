# TravelMate

TravelMate to aplikacja desktopowa wspomagająca planowanie, organizację oraz dokumentowanie podróży.

## Funkcjonalności

- 🗺️ Interaktywna mapa (GMap.NET) – wizualizacja odwiedzonych i planowanych miejsc
- ✈️ Wyszukiwarka lotów – przeglądanie połączeń lotniczych z rankingiem i trasami na mapie
- 🏨 Zarządzanie noclegami i dojazdami – rejestrowanie szczegółów podróży
- 💰 Kontrola wydatków – analiza kosztów z przeliczaniem walut (Frankfurter API)
- 📅 Kalendarz podróży – przegląd wyjazdów według dat
- 📊 Statystyki – łączne kilometry, liczba krajów, miast, lotów, średnia ocena
- 📄 Eksport do PDF – raporty z podsumowaniem podróży
- 🤖 Rekomendacje AI (Gemini 2.5 Flash) – propozycje kierunków na podstawie preferencji, budżetu i dat
- 📍 Punkty POI (OpenTripMap API) – wyszukiwanie atrakcji turystycznych w wybranym mieście
- 🛫 Lotniska europejskie (AirportsAPI) – wizualizacja lotnisk na mapie
- 🌍 Podświetlanie odwiedzonych krajów na mapie

## Technologie

- C# / WPF / .NET Framework 4.7.2
- GMap.NET
- SQLite / Entity Framework 6
- Gemini API, OpenTripMap API, Frankfurter API, AirportsAPI, Nominatim
